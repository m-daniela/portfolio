Zefon is an instant messaging application with end-to-end encryption and support for attachments and group chats.

In order to create it, I have used React, Redux Toolkit, express.js, Firebase Firestore, Firebase Authentication, socket.io, SASS. The end-to-end encryption logic was built using the VirgilSecurity framework.

The user first registers with an email, then they can create private or group chats, with or without encryption. Each chat has a "3rd party view" toggle, so the user can see what data is available to the provider.

I have created this application while I was completing my Bachelor's degree, but I have started working on a new, updated version, with Typescript.

The Library is a role-based library app with JWT authentication, created with FastAPI, React, postgresql, SQLAlchemy and tested with pytest.

The users can be normal users, who can browse, filter, borrow, return books, check and change their profile information, admins, who have acces to other functionalities, such as accepting new users and generating reports, and contact users, who only have access to the chat and all conversations.

Normal users and admins have a single chat window and can talk to the contact users. This functionality was implemented using socket.io.

It also has a mailing feature, implemented using Celery, which automatically sends emails to the users on registration, after the borrowing period has expired, or on book checkout.

I have implemented some functionalities using Flask.

Job Aggregator is a web scraper which collects the jobs from various job boards and displayes them in a clear and simplified manner.

The user can input a list of urls that will be scraped or the HTML itself. After these are parsed, the user has access to all available jobs and advanced filters based on the current data.

The filtering system includes: common searches, filter by date, application type, company, location, general location (country, continent, economic zone), job type, and the sorting system: sort alphabetically, by date.

Since the location and company options might get large, these sections have an integrated search bar, with debounce, to narrow the results.

I have created it using React, Tailwind CSS, Flask, SQLAlchemy, sqlite3, pyautogui, beautifulsoup4, pytest.
